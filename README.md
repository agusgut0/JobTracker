# 💼 JobTracker MVC

> Una aplicación web moderna, client-side y de arquitectura limpia, diseñada para centralizar, optimizar y potenciar la búsqueda laboral en el sector IT mediante la automatización de prompts de Inteligencia Artificial.

## 🚀 ¿Por qué existe este proyecto?

Hacer un seguimiento manual de decenas de postulaciones en portales fragmentados (LinkedIn, portales de ATS, etc.) genera fricción y pérdida de tiempo. 
**JobTracker MVC** resuelve este problema centralizando las búsquedas inteligentes y automatizando la preparación de la postulación. 
A diferencia de otras herramientas, este sistema **integra flujos de IA de manera agnóstica**, permitiendo al usuario generar análisis de compatibilidad (*FIT*), 
adaptar el CV y crear cartas de presentación personalizadas en segundos, exportando los resultados directamente a formatos profesionales `.doc`.
